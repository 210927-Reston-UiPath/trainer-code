Attributes

Some tags, like your img tag use attributes to define the value of what they're presenting (img src="where the image is/image address")
The href attribute of the anchor tag, tells you where the anchor leads
You can also use certain attributes to identify your elements

    2 common ways to identify an element via attributes
  • ID - unique way to identify an element
  • Class - a way to identify a group of elements
You need identification to be able to know where to apply certain CSS styling rules as well as behavior in JS. Then again, you can also use the style attribute to apply style to your elements.

CSS

Types of Selectors

Simple Selectors

  • Tag/Element
    • use tags to identify the element
    • this affects all elements with that tag name
  • Class
    • uses the class name to style a group of elements
    • denoted by .className
  • ID
    • Uses a unique ID
    • denoted by #id
  • Pseudo-class
    • specific keyword at the end of the selector used to style elements when it is at a certain state
    • :active, :hover, etc...
  • Pseudo-element
    • used to style specific parts of an element
    • insert content before or after an element
    • style first letter or line of an element
    • ::before, ::after, ::firstline, ::firstletter, etc

Combinator Selectors

  • Child Descendant
  • Sibling
  • Attribute Selectors
  • Multiple Selectors

Wild card selector

  • Just the *
  • Styles everything, all elements with the given rules
